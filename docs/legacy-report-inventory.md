## 注意事項

- これらの **Excel ファイル本体は旧パッケージのテンプレート**で、新システムでそのまま流用すべきではない(IP上のリスク)
- ただし、**「どんな帳票を社内で運用していたか」という事実は自由に参照可能**
- 各帳票の**最終出力サンプル(実際に印刷された PDF など)**を社内から集めれば、新システムのテンプレを独自設計する際の参考になる
- 山梨支店向けに分岐したテンプレ(`作業日報_備考_山梨`)があるなど、**支店別・運用別の差異**が既にあるため、新システムでは「テンプレートを管理画面で差し替え可能」な設計が必須

## 新システムでの優先実装(REQUIREMENTS.md F13 と対応)

### Phase 1 で実装すべき帳票(高頻度)
1. 実行予算書内訳書
2. 出来高査定表
3. 作業日報
4. 労務出面一覧
5. 機械出面一覧
6. 工事台帳(費目別)
7. 取引先別明細
8. 支払額管理表

### Phase 2 で実装(月次・経営報告)
9. 完成工事一覧
10. 未成工事一覧
11. 予算実績対比
12. 工事勘定簿(全種)
13. 工事費原価報告
14. 入金管理表

### Phase 3 で実装(発注・特殊帳票)
15. 見積書、請求書、外注依頼書、資材発注書
16. 工事経歴書、相殺台帳
17. その他の特殊帳票

### 検討対象外(必要時のみ)
- 名称集計系の細分化版(統合・整理)
- 支店別カスタム(管理画面で生成できれば不要)
